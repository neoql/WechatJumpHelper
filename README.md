# JumpHelper

微信小游戏《跳一跳》作弊工具。(仅供学习交流使用，不得用于非法用途)

## 安装

### Linux

```
$ mkdir build && cd build
$ cmake ..
$ make
$ sudo make install
```

## 依赖

* [OpenCV](https://github.com/opencv/opencv)

* [JsonCpp](https://github.com/open-source-parsers/jsoncpp)

* ADB调试工具

## 使用方式

### Linux

```
$ JumpHelper
```
